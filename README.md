# Time Tools

Offline GitHub Pages app with:

- Time Zone Converter
- Event Link
- World Clock
- Time Calculator
- Schedule

## Install

Upload the files in this folder to the root of a GitHub repository and enable GitHub Pages.

Open the full project URL, for example:

`https://your-user-name.github.io/your-repository-name/`

Then:

- Mac: Safari -> File -> Add to Dock
- iPhone: Share -> Add to Home Screen

The app works offline after it has loaded once.

## Note

Time zone data comes from the browser's built-in time zone database, so the app does not need internet access for conversions.
